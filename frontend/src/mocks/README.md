# Mocks (Frontend)

Hooks e utilitários para deixar o frontend “vivo” sem backend.

- `useCampaignFilters.js`: simula filtro e ordenação na lista de campanhas da Home.
- `usePeriodFilter.js`: simula seleção de período no Financeiro e retorna dados por período.
- `useFormState.js`: helper genérico para formulários controlados (setField/reset/bind).
